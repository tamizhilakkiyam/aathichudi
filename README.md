# Aathichudi

A simple site made in svelte to display Āathichoodi (ஆத்திசூடி), a collection of 109 single-line quotations written by Avvaiyar and organized in alphabetical order.

## Contribution

This site might have some bugs, if you find some bugs or want to give us suggestions kindly open an issue. Feel free to create a pull request and contribute to the site.

## Note

The data used in the site is taken from many sources.
